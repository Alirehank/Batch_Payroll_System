# Batch PayRoll System

The Batch Payroll System is a comprehensive application designed to manage the payroll process for various types of employees. All employees, whether Employee, Hourly Employee, Salaried Emp, or Commission-Based Emp, can use the system for Authentication and to Update Info. Payroll generation begins with input from different employee types: Hourly Employees Submit Time Card, and Commission-Based Emps Submit Sales Receipt. The core functionality, managed by System Operations, is to Run Payroll, which includes Calculate Pay. This calculation process includes Deduct Union Dues for Union members and factors in Service Charges. The calculated amount is used to Generate Paycheck (on a Weekly or Monthly basis), which is delivered via Recieve Paycheck through Direct Deposit or Mail, with an option to Hold the payment. A crucial Validate step includes the entire payroll process to ensure accuracy and security.
